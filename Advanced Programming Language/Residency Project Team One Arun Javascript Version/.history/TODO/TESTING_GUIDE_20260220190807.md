## 🧪 Concurrency Testing Scenarios

### Scenario 1: Real-Time Multi-User Sync ⚡

**Objective:** Verify auto-sync detects changes from other users

**Steps:**
1. **Tab A (Arun):** Open TODO app
2. **Tab B (Tarak):** Open TODO app in new tab
3. **Tab A:** Create new task "Sync Test"
4. **Tab B:** Wait 5 seconds
5. **Expected:** Tab B automatically shows "Sync Test" without page refresh

**Verification:**
- ✅ New task appears in Tab B after ~5 seconds
- ✅ No manual refresh needed
- ✅ Browser console shows: `✅ Changes detected: ADDED - New task: Sync Test`

---

### Scenario 2: Concurrent Edit Conflict Detection 🔥

**Objective:** Verify conflicts detected when same task edited by multiple users

**Steps:**
1. **Tab A & B:** Both open with same user (Arun)
2. **Both tabs:** Click "Edit" on "Learn JavaScript Fundamentals" task
3. **Tab A:** Change text to "Learn Advanced JavaScript" → Save
4. **Tab B:** Change text to "Master JavaScript Basics" → Save
5. **Expected:** Tab B shows conflict alert

**Verification:**
- ✅ Tab B displays: `⚠️ Conflict: This task was modified by another user`
- ✅ Tab B auto-refreshes and shows Tab A's version
- ✅ Version number incremented in db.json
- ✅ Audit trail logs both changes with timestamps

### Scenario 3: Version Number Incrementing 📊

**Objective:** Verify version numbers increase on each update

**Steps:**
1. Open browser DevTools → Network tab
2. Create new task
3. Edit the task 2-3 times
4. Observe PATCH requests

**Verification:**
- ✅ First update: `"version": 2` (created at v1)
- ✅ Second update: `"version": 3`
- ✅ Each request includes: `lastModifiedBy`, `lastModifiedAt`
- ✅ Timestamps differ by edit time

**Check in DevTools:**

```json
  "text": "Updated task",
  "version": 2,
  "lastModifiedBy": 1,
  "lastModifiedAt": "2026-02-20T10:05:30Z"
}
```

## 🚨 Common Issues & Solutions

| Issue | Cause | Solution |
|-------|-------|----------|
| Tasks don't update in second tab | Polling disabled or too slow | Check DevTools → check interval, wait 5s |
| Conflict alert keeps appearing | Version mismatch | Refresh page, close other tabs, clear cache |
| New tasks don't show in other tab | Different user selected | Switch to same user, wait 5s, or refresh |
| Versions not incrementing | TodoService error | Check browser console, verify db.json format |
| Audit trail empty | Never edited that task | Create new task, make edits, check trail |
| Auto-sync not working | RefreshManager not started | Check that `startAutoRefresh()` called in initialize() |

## 🔧 Customization

### Change Polling Interval (default 5 seconds)
```javascript
// In ToDoContainer.js constructor
this.refreshManager = new RefreshManager(this.todoService, 10000) // 10 seconds
```

### Disable Auto-Sync (for testing single-user mode)
```javascript
// Comment out in initialize()
// this.refreshManager.startAutoRefresh()
```

### Manual Refresh Trigger
```javascript
// In browser console
await todoContainer.refreshLocalTodos()
```
